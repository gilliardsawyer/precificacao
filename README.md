# Precifica Licita

Sistema web simples para precificacao de produtos em licitacao publica, feito em HTML, CSS e JavaScript puro.

## O que esta versao faz

- edita itens direto na tabela
- importa itens por CSV
- exporta para Excel e PDF
- gera backup e restauracao em JSON
- separa parametros globais de excecoes por item
- valida quantidade, custo e percentuais
- mostra lucro real em valor e percentual
- salva multiplas planilhas no navegador
- inclui cabecalho da licitacao
- destaca itens com margem baixa
- possui modo enxuto e filtro de alertas
- agrupa itens por lote com subtotal
- salva historico de versoes da planilha
- sugere marcas/modelos recorrentes
- fixa cabecalho e totais da tabela

## Como usar

1. Abra `index.html` no navegador.
2. Crie ou selecione uma planilha.
3. Preencha os dados do processo no cabecalho da licitacao.
4. Ajuste os percentuais globais.
5. Cadastre os itens ou edite direto na grade.
6. Use lote, filtro de alertas e modo enxuto quando precisar.
7. Exporte para Excel, gere backup JSON ou use o modo impressao/PDF.

## Observacao

Os dados ficam salvos apenas no navegador e no computador em que o sistema for usado.
